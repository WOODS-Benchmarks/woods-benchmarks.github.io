Updates

WOODS is very much still under development. We are always looking for feedback and contributions. In the near future, we will be working towards adding more datasets from real world problems from more modalities and improving the API of WOODS.


Initial Release!

XX November 2021

WOODS is released to the public!

It includes the following benchmarks:

  • Spurious_Fourier
  • Temporal Colored MNIST Sequences
  • Temporal Colored MNIST Steps
  • CAP
  • SEDFx
  • MI
  • LSA64
  • HAR

With the following OOD objectives:

  • ERM
  • IRM
  • VREx
  • Spectral Decoupling (SD)
  • ANDMask
  • Fish

Stay tuned for more updates! More datasets will be added soon!
